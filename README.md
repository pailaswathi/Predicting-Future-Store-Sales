# Predicting-Future-Store-Sales
Time Series Forecasting project that predicts future store sales using statistical analysis and the ARIMA Model after performing stationarity tests, log transformation, differencing, and ACF/PACF analysis.

This project focuses on forecasting future store sales using time series analysis techniques. The dataset is analyzed to identify patterns, trends, and seasonality, and then a forecasting model is built to predict future sales.

The project uses statistical techniques such as stationarity testing, data transformation, and autocorrelation analysis before applying the forecasting model.

Project Objectives:

Analyze historical store sales data

Identify trends and patterns in the data

Convert the data into a stationary time series

Use statistical tests to validate stationarity

Build a forecasting model to predict future sales
